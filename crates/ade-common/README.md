# Ade-common

`ade-common` provides common utilities and test helpers.

## Installation

Add this to your `Cargo.toml`:

```toml
[dependencies]
ade-common = "0.1.0"
```

## Documentation

The complete documentation is available on [docs.rs](https://docs.rs/ade-common).
